# arduino-april-fools-script
An Arduino script for April Fools.

Use this on April Fools to scare someone.

# Required:
Arduino IDE

Arduino Nano (or Arduino Uno)

Breadboard

Jumper Wires

HC-SR04

Active Buzzer

Portable USB Power Bank (optional)

USB cable
# Wiring:
HC-SR04 VCC > Arduino 5V

HC-SR04 Trig > Arduino D6

HC-SR04 Echo > Arduino D5

HC-SR04 GND > Arduino GND

Buzzer + > Arduino D3

Buzzer - > Arduino GND

# Instructions:
1. Wire it on the breadboard with the provided wiring.

2. Plug it into your computer.

3. Download and install Arduino IDE.

4. Open script.ino.

5. Upload it.

If you have a power bank, do this:

1. Unplug it from your computer.

2. Plug it into the power bank.

3. Take it anywhere you want.
